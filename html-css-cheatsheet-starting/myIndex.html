CSS External Stylesheet Composition Cheatsheet

Selectors

Selector in Html Html Example in CSS CSS Example How It Works
Universal none none * * {
font-family: Verdana;
}		 To apply a rule to every site element,
use the universal selector at the top of
the stylesheet.
Type [<]tag-goes-here[>] [<]body[>] tag without [<][>] body {
}		 To select major sections for styling,
use the html type element without prefixes.
Class [<]class="class-goes-here"[>] [<]div class="green-box"[>] .class-goes-here .green-box {
}		 To select a class element for styling,
start with a period, then
use the contents of the html class element
listed between quotes.
Multiple Classes [<]class="class-goes-here, another-class-goes-here"[>] [<]div class="green-box, outline-box"[>] .class-goes-here{
}
.another-class-goes-here{
}		 .green-box {
}
.outline-box {
}		 For a selector with more than one class,
styling can be done separately by listing
all the classes on their own in the CSS.
ID [<]id="id-goes-here"[>] [<]div id="single-element"[>] #id-goes-here #single-element {
}		 To select an id element for styling,
start with a hashtag, then
use the contents of the html id element
listed between quotes.
Attribute [<]element class="specific-example"[>] [<]img src="summer"[>] element [class*='specific-example'] img [src*='summer'] {
}		 Attributes can be selected by their characters
by treating it as SQL notation.
Brackets enclose the string to select, and asterisk
indicates "select all".
