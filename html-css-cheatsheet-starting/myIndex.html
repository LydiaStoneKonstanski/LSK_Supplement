Selectors

Selector in Html Html Example in CSS CSS Example How It Works
Universal none none * * {
font-family: Verdana;
}		 To apply a rule to every site element,
use the universal selector at the top of
the stylesheet.
Type [<]tag-goes-here[>] [<]body[>] tag without [<][>] body {
}		 To select major sections for styling,
use the html type element without prefixes.
Class [<]class="class-goes-here"[>] [<]div class="green-box"[>] .class-goes-here .green-box {
}		 To select a class element for styling,
use the contents of the html class element
listed between quotes.
